# Didula Sri Lakpriya - Portfolio Website

A modern, responsive portfolio website built with Next.js 15, TypeScript, and Tailwind CSS. Features dark/light mode toggle, smooth animations, and a clean, professional design.

## 🚀 Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Dark/Light Mode**: Toggle between dark and light themes with persistent preference
- **Modern Tech Stack**: Built with Next.js 15, React 18, TypeScript, and Tailwind CSS
- **Smooth Animations**: Engaging animations and transitions throughout
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Performance**: Optimized for speed and performance
- **Accessible**: Built with accessibility best practices

## 📋 Sections

1. **Home**: Hero section with animated text and social links
2. **About**: Profile summary, highlights, and technical skills
3. **Experience**: Professional experience and education
4. **Projects**: Featured projects with descriptions and links
5. **Connect**: Contact information and message form

## 🛠️ Technologies Used

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Fonts**: Google Fonts (Syne, JetBrains Mono, Space Mono)

## 📦 Installation

1. **Clone the repository** (or extract the files):
   ```bash
   cd didula-portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser** and navigate to:
   ```
   http://localhost:3000
   ```

## 🔧 Configuration

### Customization

You can customize the portfolio by editing the following files:

- **Personal Info**: Update content in component files under `/components`
- **Colors**: Modify color scheme in `tailwind.config.ts`
- **Projects**: Edit project data in `/components/Projects.tsx`
- **Experience**: Update experience details in `/components/Experience.tsx`
- **Contact**: Modify contact information in `/components/Connect.tsx`

### Color Scheme

The portfolio uses a modern, tech-focused color palette:
- **Primary**: `#00ff88` (Neon Green)
- **Accent Cyan**: `#00ffff`
- **Accent Purple**: `#a855f7`
- **Accent Pink**: `#ec4899`
- **Dark Background**: `#0a0a0a`
- **Light Background**: `#ffffff`

## 📱 Deployment

### Build for Production

```bash
npm run build
```

### Start Production Server

```bash
npm start
```

### Deploy to Vercel

The easiest way to deploy is using [Vercel](https://vercel.com):

1. Push your code to GitHub
2. Import the repository in Vercel
3. Deploy with one click

Alternatively, you can deploy to:
- Netlify
- AWS Amplify
- Google Cloud Platform
- Any platform that supports Next.js

## 🎨 Design Philosophy

This portfolio follows these design principles:

1. **Bold & Modern**: Uses a striking neon color palette with dark/light themes
2. **Tech-Focused**: Incorporates tech symbols and code aesthetics
3. **Clean & Professional**: Maintains readability with proper spacing and typography
4. **Interactive**: Smooth animations and hover effects for engagement
5. **Unique**: Avoids generic AI aesthetics with custom fonts and layouts
